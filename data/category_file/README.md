## рџ†• Adding a New Category

To add a new category **without changing any code**:

1. **Create category folder** in `data/`:
   ```bash
   mkdir data/your_new_category_name
   ```

2. **Add PDF files** to the new folder:
   ```
   data/your_new_category_name/
   в”њв”Ђв”Ђ document1.pdf
   в”њв”Ђв”Ђ document2.pdf
   в””в”Ђв”Ђ document3.pdf
   ```

3. **Update categories.csv** - Add a new line:
   ```csv
   your_new_category_name,Your Category Display Name,true
   ```

4. **Restart the API** - Categories are loaded at startup

5. **Test the new category**:
   ```bash
   curl -X POST https://your-api-url/generate-mcqs \
     -H "Content-Type: application/json" \
     -d '{"category": "your_new_category_name"}'
   ```

## рџ”„ Deactivating a Category

To temporarily disable a category without deleting it:

1. Open `categories.csv`
2. Change the `active` value from `true` to `false`
3. Restart the API

```csv
# Before (active)
circular_economy_and_waste_reduction,Circular Economy and Waste Reduction,true

# After (inactive)
circular_economy_and_waste_reduction,Circular Economy and Waste Reduction,false
```

## вљ пёЏ Important Notes

- **Category names must match folder names** in the `data/` directory
- Category names should use **lowercase** and **underscores** (e.g., `my_category`, not `My Category`)
- The API validates that categories exist in the CSV before generating MCQs
- Changes to `categories.csv` require an **API restart** to take effect
- Keep the CSV header row: `category_name,description,active`
